# Computer-Buying-Help
Edcuational website on buying computers with tools such as chatbots and links to helpful addtional resources to help decide on the best configuration across different use cases. This site does not suggest any specific manufacturer, model, or provide pricing information, but teaches the principles behind finding the best setup for the money.

## Features
- Minimalist, modern UI design with animations and page transitions
- Mobile-friendly
- Two chatbots to help with general questions and narrowing down the ideal set of features respectively for a new (or used/refurbished) computer
- List of community, video, and web resources for reliable information on the latest hardware releases, additional details about the buying process, or customizing a setup to better suit a specific use case

## Technologies
- Next.js
- TailwindCSS
- TypeScript
- Google Dialogflow chatbot
- Typebot chatbot

## Getting Started
